# Yemek Sayfası

Bu proje, yemek tariflerini ve öğün önerilerini sergileyen basit bir web sitesidir. HTML ve CSS kullanılarak geliştirilmiştir ve responsive tasarımı sayesinde farklı cihazlarda sorunsuz çalışır.


---

## Kurulum

Projeyi kullanmak için öncelikle GitHub reposunu bilgisayarınıza klonlayın: `git clone https://github.com/belinaykaraasln-beep/Yemek-Sayfasi.git`. Daha sonra proje dizinine girin: `cd Yemek-Sayfasi`. Siteyi tarayıcıda açmak için `index.html` dosyasını çift tıklayabilir veya terminalden Mac için `open index.html`, Windows için `start index.html` komutunu kullanabilirsiniz.

---

## Kullanım

- Ana sayfada yemek tarifleri ve öğün önerileri görüntülenir.  
- CSS ile tasarlanmış modern ve temiz bir arayüz sağlar.  
- Tarifleri ve içerikleri güncellemek için HTML ve CSS dosyalarını düzenleyebilirsiniz.
